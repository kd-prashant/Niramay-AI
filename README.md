# Niramay AI: Healing the earth, one leaf at a Time 🌿

[![Tech Stack](https://img.shields.io/badge/Stack-Next.js%20|%20FastAPI%20|%20Supabase-green)](https://github.com/yourusername/Niramay-AI)
[![UI/UX](https://img.shields.io/badge/Design-Clinical--Organic-zinc)](https://niramay-ai.vercel.app)

**Niramay AI** is a state-of-the-art agricultural diagnostic platform designed to bridge the gap between advanced bio-neural science and the everyday farmer. Using high-fidelity computer vision and edge-computing, it provides instant, clinical-grade assessments of crop health.

## ✨ Key Features

- **Rapid AI Assessment**: Native-feeling mobile interface that provides diagnostic results in under 2 seconds.
- **Biosphere Monitor**: Real-time environmental tracking (Humidity, Temperature, and Blight Risk) powered by live meteorological data.
- **Bilingual Interface**: Seamless support for **English** and **Hindi**, ensuring accessibility for the heart of the farming community.
- **Infinite Telemetry History**: A real-time archive drift that tracks every health assessment for long-term field monitoring.
- **Clinical-Organic Branding**: A premium high-contrast UI (Deep Zinc & Veridian Green) designed for maximum legibility in outdoor agricultural conditions.

## 🛠️ Technology Stack

- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion (for cinematic animations), Lucide React.
- **Backend**: FastAPI (Python), TensorFlow/PyTorch (AI Inference).
- **Database & Auth**: Supabase (PostgreSQL).
- **APIs**: OpenWeatherMap (Meteorological data).

## 🚀 Installation & Setup

### Prerequisites
- Node.js (v18+)
- Python (3.9+)
- Supabase Account
- OpenWeatherMap API Key

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn api:app --reload
```
